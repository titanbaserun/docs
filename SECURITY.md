# Security Policy

## Reporting a vulnerability

If you discover a security issue in the Titanbase documentation site or its
infrastructure, please report it privately.

- Email: **hi@titanbase.run**
- Or use GitHub's [private vulnerability reporting](https://github.com/titanbaserun/docs/security/advisories/new).

Please do **not** open a public issue for security reports.

Include as much detail as possible: a description of the issue, steps to
reproduce, and the potential impact. We'll acknowledge your report and keep you
updated on the fix.

## Scope

This repository contains documentation content and a static Next.js site. For
vulnerabilities in the Titanbase product itself (editor, core, exporters, CLI),
report them at [titanbaserun/titanbase](https://github.com/titanbaserun/titanbase/security).
