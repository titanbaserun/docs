# Desktop App

<Status value="Planned" />

A Titanbase desktop app is on the roadmap. **It is not available yet.**

## The plan

- **Free and open source**, like the rest of the local product.
- Packages the same **local-first editor** you use in the browser.
- **No account and no cloud backend** required.
- Opens and saves `.titan.json` files directly from your filesystem.

<Callout type="note">
The desktop app is about convenience — a native window over the local editor — not new capabilities behind an account. It will keep the same promise: your schema stays local.
</Callout>

Until it ships, use the [browser editor](https://app.titanbase.run) or [run it from source](/contributing). When the desktop app is available, this page will switch to **Status: Available** with download instructions.
